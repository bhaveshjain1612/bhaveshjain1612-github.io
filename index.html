My Project - StockSpectra

**StockSpectra** is an open-source web application that assists investors by analyzing stocks traded on Indian markets. I developed StockSpectra to provide a full suite of tools for technical and fundamental analysis, peer comparisons, and stock picking insights.

**Key Results from StockSpectra** that showcase my data skills:

  • 66.94% Win Rate: StockSpectra identified stocks with a win rate of 66.94% based on historical data, highlighting my proficiency in predictive analytics.
  • Median Return of 5.6%: Achieved through careful mathematical modeling and backtesting, providing reliable insights for investors.
  • Data-Driven Optimization: My analysis resulted in an average win return of 11.83%, outperforming loss returns (-7.88%) and demonstrating robust risk management skills.
  • Efficient Data Visualization: Developed intuitive visual tools to track metrics like average holding period (30.78 days) and median holding period (19 days), making data easy to interpret for non-technical stakeholders.
